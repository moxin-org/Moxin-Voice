# TTS 输出音色选项

本文档说明前端“输出音色”选项与请求参数的映射规则。

## 参数说明

- 语速（`speed`）
  - 慢：`0.85`
  - 中：`1.00`
  - 快：`1.15`
- 音调（`pitch`，单位：半音）
  - 低：`-3`
  - 中：`0`
  - 高：`+3`

前端会把参数打包到统一前缀：

`TTSCFG|<speed_factor>|<pitch_semitones>|<VOICE payload>`

其中 `<VOICE payload>` 继续沿用原有 `VOICE:*` 协议（内置/自定义/训练音色）。

## 默认行为

- 默认语速：中（`1.00`）
- 默认音调：中（`0`）
- 如果用户未主动切换，发送请求时仍会携带默认值，保证参数完整。

## 回退与边界

后端会对参数进行边界约束：

- `speed_factor` 限制在 `[0.5, 2.0]`
- `pitch_semitones` 限制在 `[-12, 12]`

当 `TTSCFG` 解析失败或值非法时：

- 语速回退到当前语音配置的默认速度
- 音调回退到 `0`（不偏移）

## 已知限制

- 当前音调处理采用轻量重采样方式，可能带来轻微音色失真，并可能改变片段时长。
- 不同声线模型对语速/音调的感知强度可能不同，主观听感会有差异。
