## Why

当前 `select voice` 的声音特质分类不再匹配业务目标：`童声` 已不需要保留，同时缺少更贴近实际使用场景的分类。为提升选音效率和角色一致性，需要立即调整分类结构并补充指定角色音色。

## What Changes

- 从 `select voice` 中移除 `童声` 分类及其入口。
- 新增两个声音特质分类：`专业播音`、`特色人物`。
- 在 `专业播音` 分类下新增并展示：`白岩松`、`罗翔`、`沈逸`。
- 在 `特色人物` 分类下新增并展示：`马云`、`杨幂`、`周杰伦`、`trump罗翔`。
- 调整分类与声音项的数据映射和渲染逻辑，确保前端展示与数据源一致。

## Capabilities

### New Capabilities
- `voice-trait-category-management`: 管理 `select voice` 中声音特质分类与分类下声音清单（新增分类、删除分类、分类内声音配置）。

### Modified Capabilities
无（当前无既有 capability 规格文件）。

## Impact

- 影响模块：`select voice` 相关 UI、声音分类配置、分类筛选逻辑。
- 影响数据：声音特质分类枚举与分类-声音映射表。
- API 影响：如分类来自后端配置，需同步更新返回结构或静态配置源；若为本地常量，需更新常量定义与消费处。
